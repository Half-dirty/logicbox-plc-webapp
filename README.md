# LogicBox PLC WebApp

Курсовой проект по дисциплине «Разработка веб-приложений».

Тема проекта: разработка веб-приложения для управления конфигурациями и диагностическими данными ПЛК LogicBox в системах АСУТП.

## Ссылки

- Демонстрационный сайт: https://logicbox-web-halfdirty.waw0.amvera.tech/
- Репозиторий: https://github.com/Half-dirty/logicbox-plc-webapp

## Назначение

Система предназначена для централизованного учета объектов эксплуатации, контроллеров LogicBox, конфигураций, диагностических записей, файлов и сессий синхронизации. Приложение не выполняет удаленное управление ПЛК и не отправляет управляющие команды на оборудование.

## Основные возможности

- аутентификация пользователей и проверка активной сессии;
- ролевая модель доступа: администратор, инженер, ведущий инженер, наблюдатель;
- разграничение прав по объектам эксплуатации;
- иерархия объектов: организация, участок, стенд, шкаф;
- учет ПЛК LogicBox с привязкой только к шкафам;
- CRUD-интерфейсы для объектов, контроллеров, конфигураций, файлов, пользователей и локальных клиентов;
- загрузка, скачивание и безопасное хранение файлов;
- архивация ПЛК со связанными конфигурациями, диагностикой и файлами;
- прием данных от локального клиента через API по Bearer-токену;
- журнал аудита и аналитические страницы.

## Технологии

- Backend: Go, Fiber.
- Database: PostgreSQL.
- Frontend: HTML, CSS, JavaScript.
- Deployment: Docker, Docker Compose, Amvera.
- Report: LaTeX.

## Структура репозитория

```text
logicbox_webapp_ready/   исходный код веб-приложения
  cmd/web/               точка входа приложения
  internal/app/          серверная логика, маршруты, доступ, CRUD, API
  internal/config/       конфигурация приложения
  internal/database/     подключение к БД, миграции, начальные данные
  internal/storage/      файловое хранилище
  migrations/            SQL-миграции PostgreSQL
  web/templates/         HTML-шаблоны интерфейса
  web/static/            CSS, JavaScript и статические файлы
images/                  изображения для отчета
*.tex, *.bib, *.sty      исходники отчета в LaTeX
paper.pdf                итоговый PDF-отчет
```
