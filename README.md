# logicbox-plc-webapp

Курсовой проект по дисциплине «Веб-приложения».

## Тема проекта
Разработка веб-приложения для централизованного учета, хранения, анализа и синхронизации конфигурационных и диагностических данных ПЛК LogicBox в системах АСУТП.

## Назначение системы
Проект ориентирован на создание веб-приложения, которое позволяет:
- вести реестр объектов автоматизации и контроллеров LogicBox;
- хранить конфигурации, диагностические данные и сопроводительные файлы;
- разграничивать доступ пользователей по ролям и зонам ответственности;
- агрегировать и анализировать накопленные сведения по объектам и устройствам;
- принимать данные от внешнего локального клиента сбора данных без реализации опасного удаленного управления ПЛК через веб-интерфейс.

## Ключевая идея
Основной объект разработки в рамках курсового проекта — именно веб-приложение.

Локальный клиент рассматривается как внешний интеграционный компонент, который может:
- работать в локальной сети объекта;
- собирать сведения о ПЛК LogicBox;
- передавать результаты на сервер веб-приложения для дальнейшего хранения и анализа.

## Планируемые функции веб-приложения
- аутентификация и авторизация пользователей;
- ролевая модель и разграничение прав доступа;
- CRUD-интерфейс для объектов, контроллеров, конфигураций и диагностических записей;
- загрузка и выгрузка файлов;
- фильтрация, группировка и агрегирование данных;
- прием данных от локального клиента по защищенному API.

## Предварительный стек
- Backend: Go
- Frontend: HTML, CSS, JavaScript
- Database: PostgreSQL
- Отчёт: LaTeX

## Ограничения базовой версии
В базовой версии проекта не реализуется удаленное управление ПЛК через веб-интерфейс.
Система предназначена для безопасного учета, хранения, анализа и синхронизации данных.
